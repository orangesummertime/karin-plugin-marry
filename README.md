# 👫 Karin 群对象插件（MarryGroupFriends）

> 一个为 **Karin Bot** 打造的趣味群聊插件，让群员可以 “强娶 / 抢群友 / 闹离婚 / 查询对象 / 娶群友（有bug待修复）”。  
> 兼容多适配器，逻辑干净无副作用，支持分群独立存储与自动清理过期关系。

---

## 🧩 支持的命令

| 指令 | 说明 |
|------|------|
| `#娶群友` | 随机抽取一位群友成为对象（30% 概率失败） |
| `#强娶 @对象` | 强行娶指定群友，若已有对象则提示 |
| `#抢群友 @对象` | 有 30% 概率成功从别人手里抢走对象 |
| `#闹离婚` | 与当前对象解除关系 |
| `#我对象呢` | 查询自己当前的群内对象 |
| `#群对象列表` | 查看当前群的所有恋爱关系列表（附结婚时间） |

---

## 🧭 使用说明

1. 将插件文件放入你的 Karin 插件目录中，例如：
@karinjs/plugins/marry-group-friends/index.js
2. 启动或重启 Karin，即可自动加载。
3. 在群聊中发送上表指令即可体验。

---

## 🔧 自动清理机制

- 每段婚姻维持时间为 **24 小时**（可修改 `DAY_MS` 的值调整）。
- 触发任意命令或每 30 分钟的后台定时任务都会执行清理：
- 若记录超过 24 小时，将自动解除关系。
- 控制台日志会打印清理数量。

---

## 🕐 时间与时区

所有展示时间均以 **北京时间（GMT+8）** 输出，  
逻辑计算仍基于 UTC 时间戳，确保跨时区机器稳定运行。
